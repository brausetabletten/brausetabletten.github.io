# AntiGrav Racer

This is the project I am currently working on: a hover/anti-gravity racing game. It is still in very early phases, resembling more of a math problem than an actual game. Still, there is plenty to blog about already. 

I felt motivated to prototype an AG racing game since I tried out the F-zero games via the Switch's N64 / GC collections.
